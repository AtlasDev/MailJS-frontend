## MailJS frontend

This is the new version of the MailJS frontend, build from the ground up.

This repository contains the source code for development. You need build tools to create a usable frontend. Please check the release tab for a precompiled version.


### Usage
``grunt build``:

Do a full build of the app, takes a while to complete. A zip file will be found in `/builds` containing the builded file.

``grunt dev``:

Watch files for changes, and rebuild the app when a change appears. Development only

``grunt hint``:

Checks the JavaScript for errors.
